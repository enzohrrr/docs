---
title: "Setup"
---

<Steps>

  <Step title="First Step">
    Create a **Blueprint Class** based on `DGC_CameraActor`.
  </Step>

  <Step title="Second Step">
    Create a **Data Asset** based on `DGC_CameraData` to configure camera behavior.

    <Frame>
      <img src="/plugins/dynamic-gameplay-camera/assets/class-selection.png"/>
    </Frame>

    <Note>Creating a `DGC_CameraData` asset is optional. If no Data Asset is assigned, the plugin will automatically create a temporary instance at runtime using the default values defined in the base C++ Data Asset.</Note>

    This Data Asset centralizes all camera settings and allows you to enable or disable individual features.
  </Step>

  <Step title="Third Step">
    Open your `DGC_CameraActor` Blueprint and, in the **Details** panel, assign your `DGC_CameraData` asset under the **Configuration** category.
    
    <Frame>
      <img src="/plugins/dynamic-gameplay-camera/assets/data-asset-assignement.png"/>
    </Frame>
    
  </Step>

</Steps>
